DSS 2026 Prototype

Agent-First Submissions

A lightweight submission flow inspired by agent-native conference ideas: authors can start from the terminal, register once, then submit structured talks/workshops.

Read The Canonical Spec

All registration + submission instructions are in one markdown file:

curl -fsSL https://fishbiscuit.github.io/dss-submit-test/submit.md

What The Prototype Does

  1. Defines required registration fields and IDs.
  2. Defines required submission schema and allowed session formats.
  3. Provides a direct webhook submission payload template.
  4. Acts as the single source of truth for DSS prototype intake.
